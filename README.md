# Educational-Platform-Development

This project is a full-stack web application designed to manage student attendance for an online education platform. It provides a system for tracking students, instructors, courses, and lessons. The application allows for full CRUD (Create, Read, Update, Delete) operations on student records and includes features to list course details, view attendance reports, and search for specific lessons or exams.

Technologies Used:

- Backend: Python, Flask
- Database: MySQL
- Frontend: HTML (rendered via Flask's render_template)
- Connector: mysql-connector-python
